### Prerequisites

You have to install a couple of softwares.

* gcc
  ```sh

  ```

  ```sh
  sudo apt-get install build-essential
  ```
  
### Installation

Below is an example of how you can instruct your audience on installing and setting up your app.


  ```sh
  sudo apt-get install gcc
  ```

